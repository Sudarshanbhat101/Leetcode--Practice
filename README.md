# LeetCode Practice Journey

This repository tracks my daily LeetCode progress in a clean, consistent format for interview preparation and long-term problem-solving growth.

## Goal

- Build strong DSA and problem-solving fundamentals.
- Stay consistent with daily coding streaks.
- Prepare for software engineering interviews with structured revision.

## Progress Strategy

- Push solutions daily in chronological order.
- Add only solved problems for the day.
- Keep each day small, neat, and review-friendly.
